Projects

Norm app

This app fetches data from the twitter API to display the recent tweets made about Norm Macdonald, as a tribute to the late comedian. During the process, I encountered a CORS issue for the first time. To solve the issue, I served the React app from an express server; allowing the server to make the API request. Through the building of Norm, I discovered Express, Node, CORS, working with an external API, and page-designing.

Whose Round

This drink tracking app was the result of a casual pub chat, where we discovered that it was difficult to track rounds with mates. The app allows users to enter their name, date, and the quantity of their purchased drink, where users are also able to access their rounds history.
